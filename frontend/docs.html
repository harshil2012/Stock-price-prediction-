Data Acquisition

Major Minor utilizes the yfinance library as its primary ingest engine for historical and real-time market data. We implement a custom wrapper to handle rate-limiting and session persistence across high-frequency requests.

python // acquisition_wrapper.py 
import yfinance as yf

def fetch_market_snapshot(ticker):
    data = yf.Ticker(ticker)
    # Pulling 1y history with daily intervals
    history = data.history(period="1y", interval="1d")
    return history.dropna()

Data Normalization

To ensure model convergence, all financial metrics (Volatility, RSI, Volume) are mapped to a unified 0-1 scale using Scikit-learn's MinMaxScaler. This prevents large-value features from dominating the gradient descent path during optimization.

Technical Insight

"The normalization layer acts as the 'Great Leveler', allowing our neural nets to weigh a $50,000 price movement with the same mathematical precision as a 0.5% sentiment shift."
